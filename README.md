## 项目地址

```shell
git clone https://github.com/Ikarows/cos-template.git
```

<br />

## `node` 版本推荐

因为该模板完全面向现代，所以推荐大家使用 `node` 当前的长期维护版本 `v16`， 大于 `v16.13.1` 即可。
<br />

## 技术栈

- Vue3
- Vite3
- TypeScript
- VueRouter4
- Axios
- Pinia
- Eslint
- Mocks

## 开发

```bash

# install cnpm
npm install cnpm -g --registry=https://registry.npm.taobao.org;

# install dependencies
cnpm i

# strat
npm run serve

# build
npm run build
```
